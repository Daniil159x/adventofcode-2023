# Advent of Code 2023

Site of competition: https://adventofcode.com/2023

## Goal

The main goal - learn the rust.  

## How to develope

0. Clone repository
1. Open vscode
2. Open folder inside devcontainer
3. Coding!

## How to run

### Run tests

```bash
# all tests
cargo test

# specific day
cargo test -p day01
```

### Solve challenge

Solve the first day challenge:
```bash
cat day01/input/challenge.txt | cargo run day01
```

### Benchmark

```bash
cargo bench
```
